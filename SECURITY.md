# Security Policy

## Reporting a Vulnerability

如果你发现 FindMyBV 会泄露用户信息、上传 Cookie、误操作 B 站数据，或存在其他安全问题，请先通过私下渠道联系维护者，不要直接公开可利用细节。

## Security Notes

- 项目不需要服务端密钥。
- 书签脚本不应上传 Cookie、账号凭据或完整稿件数据。
- 提交 issue 或 PR 时不要附带真实 Cookie、`SESSDATA`、`bili_jct`、浏览器导出的账号数据或私有接口响应。

## Supported Versions

当前项目处于早期版本，默认只维护最新版本。
